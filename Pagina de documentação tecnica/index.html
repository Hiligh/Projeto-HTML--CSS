Por onde começar

O inicio de toda a aplicação Java é a função main.

É definidar por (String[] arg) como o parametro da função. 

                
                    public class MyMainFunction {
                        //Exemplo de uma função main
                        public static void main(String[] args){

                        } 
                    }

Como pode ser visto a função main fica dentro da classe, e isso é uma parte fundamental da Programação orientada a objetos dos projetos em Java.

ou seja o nome do projeto é "MyMainFunction"

Imprimindo no console

Para imprimir no console, utilizamos system.out.println.

Parece complicado mas é assim que se faz. 

                
                    public class MyMainFunction{
                        public static void main(String [] args){
                            system.out.println("Hello World");
                        }
                    }

Nesse exemplo estamos imprimindo "Hello World" no console quando rodamos o programa.

Programação orientada a objetos

Java é conhecida como uma linguagem orientada a objetos.

Isto significa que é facil representar entidades como objetos usando classes e encapsulamento.

Um exemplo disso seria a classe estudande para representar um estudante 

                
                    public class Student{
                        //propriedades da classe
                        private String name;
                        private int age;

                        //construtor
                        public Student(String name, int age){
                            this.name = name;
                            this.age = age;
                        }
                        //metodo getter
                        public String getName(){
                            return name;
                        }
                        //metodo setter
                        public void setName(String name){
                            this.name = name;
                        }
                    }

Usamos essa classe fazendo o seguite:

                
                    Student student1 = new Student("joao",16);
                    String joaoName = student1.getName();
                    student1.setName("paulo");
                    String pauloName = student1.getName();
